# 微前端框架——乾坤测试

## 到各自工程下
```
yarn install
```

### 启动服务
```
yarn serve
```

### 打包
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

## 访问地址
```
http://localhost:9001/child/
http://localhost:9000
```
